# Documentação do Projeto: Game Design

## 1. Visão
O projeto visa criar um jogo de plataforma focado em narrativa e exploração, onde o jogador deve resolver mistérios num mundo futurista.

## 2. Requisitos Funcionais (RF)
- **RF01:** O sistema deve permitir que o jogador salte e use um "dash".
- **RF02:** O sistema deve ter um inventário para itens colecionáveis.
- **RF03:** O jogo deve salvar automaticamente nos checkpoints.

## 3. Requisitos Não Funcionais (RNF)
- **RNF01:** O jogo deve correr a 60 FPS estáveis.
- **RNF02:** O tempo de carregamento entre níveis não deve passar de 5 segundos.

## 4. Casos de Uso
- **UC01: Coletar Item:** O jogador aproxima-se do item, pressiona 'E' e o item vai para o inventário.
- **UC02: Morrer:** Ao tocar num espinho, o jogador volta ao último checkpoint.

## 5. Fluxo de Jogo
Menu Inicial -> Tutorial -> Nível 1 -> Boss -> Créditos.
